# webapp

## Requirements

- Node.js v22
- Yarn package manager
- nvm (Node Version Manager)

## Setup

### Install Node.js v22 using nvm

```bash
  nvm install 22
  nvm use 22
```

### Install Yarn

If you don't have Yarn installed, you can install it using npm:

```bash
  npm install -g yarn
```

## Deployment

### Package Installation

```bash
  yarn install
```

### Running with development server

```bash
  yarn run dev
```

### Building production

```bash
  yarn run build
```
